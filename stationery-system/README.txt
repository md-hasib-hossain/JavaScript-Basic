Stationery Management System Backend - Step 1 & 2
======================================================

কীভাবে এটি চালু করবেন:
১. জিপ (Zip) ফাইলটি আনজিপ করুন।
২. আপনার XAMPP Control Panel চালু করে 'MySQL' স্টার্ট করুন।
৩. ব্রাউজারে http://localhost/phpmyadmin এ গিয়ে 'stationery_db' নামে একটি নতুন ডাটাবেজ তৈরি করুন।
৪. আপনার টার্মিনাল/কমান্ড প্রম্পটে এই ফোল্ডারের ভেতর প্রবেশ করুন (cd stationery-system)।
৫. নিচের কমান্ডটি দিয়ে প্রয়োজনীয় প্যাকেজগুলো ইন্সটল করুন:
   npm install
৬. প্রজেক্টটি চালু করতে রান করুন:
   npm start

সফলভাবে রান হলে ডাটাবেজে স্বয়ংক্রিয়ভাবে 'users' এবং 'categories' টেবিল দুটি তৈরি হয়ে যাবে।
